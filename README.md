## Run Locally

Clone the project

```bash
  git clone https://github.com/CodeXtecch/grow-geeky
```

Go to the project directory

```bash
  cd grow-geeky
```

Install dependencies

```bash
  npm install
```

Start the development server

```bash
  npm run dev
```
